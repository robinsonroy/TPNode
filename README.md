## Introduce
This a nodeJS project to learn :
  - Best practices
  - Express
  - Middleware
  - Coffee-script / Pug Transpilers
  - Socket with socket.io
  - d3chart library
  - Unit testing with Should / Mocha / Travis
  - Key Value database with LevelDB

---
## Install
```
npm install -g nodemon
npm install
bin/populated
bin/start
```
